# 研究员学术主页
基于 GitHub Pages 部署，分页面设计，方便后期维护。

## 页面结构
1. index.html - 首页（个人简介+页面导航）
2. research-direction.html - 研究方向
3. research-achievements.html - 科研成果
4. team-members.html - 团队成员
5. contact-us.html - 联系我们

## 部署方法
1. 新建 GitHub 仓库，命名为：用户名.github.io
2. 将上述5个文件（index.html、research-direction.html、research-achievements.html、team-members.html、contact-us.html、README.md）上传至仓库根目录
3. 进入仓库 Settings → Pages → 选择 main 分支，等待1-2分钟即可访问
4. 访问地址：https://用户名.github.io

## 维护说明
- 各页面独立，可单独修改某一页面内容，不影响其他页面
- 导航栏已统一配置，修改页面名称时需同步更新所有页面的导航链接
- 团队成员、论文等信息可直接在对应页面编辑修改
